# Acesso ao projeto LogicGame
* Você pode baixar o arquivo zipado LogicGame
# Abrir e rodar o projeto
* Após baixar o projeto zipado Você pode abrir a pasta com o Visual Studio. Para isso clique em:
- Descompactar a pasta .ZIP 
- Abrir uma pasta (ou alguma coisa similar)
- Procure o local onde a pasta está (precisa Descompactar o ZIP antes)
- Clique em selecionar pasta
- Irá abrir o código do em C# aperte no botão Documento Atual (Program.cs)
- Se não tiver o botão deve haver os arquivos do projetos na direita selecione Progam.cs e siga o item anterior
